# wiki/sources/ — one summary page per ingested source

The LLM writes one page here for each source ingested from [`../../raw/`](../../raw/). A source page captures the key takeaways, notable quotes (with location), how the source maps to POM concepts/principles, and any claims that **confirm or contradict** existing wiki pages.

Use the [`source-summary`](../../templates/source-summary.md) template. Filename mirrors the raw source slug.

Source pages are the connective tissue: every claim on a concept or principle page should trace back to one or more pages here via `sources:` frontmatter and inline citations.
