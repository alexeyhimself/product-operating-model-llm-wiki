# wiki/case-studies/ — transformations and real examples

One page per concrete example: a company transformation story, a team that moved from feature-factory to empowered, a product decision and its outcome. Include both SVPG-published cases and **your own organization's** examples as you accumulate them — the latter are the most valuable for coaching.

Use the [`case-study`](../../templates/case-study.md) template.

A good case study: context / starting state, what changed, which [`principles`](../principles/) and [`concepts`](../concepts/) it illustrates, the outcome, and the transferable lesson.
