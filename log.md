## [2026-06-15] revise | Tooling note: avoid recursive root globs
Added a warning to `CLAUDE.md` §6 (Tooling notes): a recursive glob/listing from the repo root (e.g. `**/*`) can be flooded by `.git/objects`/pack files (often the most-recently-modified files after a commit/repack), pushing `index.md`, `log.md`, `backlog.md` and the populated `wiki/` out of a capped, mtime-sorted result. This previously caused an agent to misreport the repo as an empty scaffold. Agents should orient via `index.md`/`log.md` directly or use scoped globs (`*.md`, `wiki/**/*.md`).
